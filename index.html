Complete SideStore Installation Guide

Follow these comprehensive steps to install and configure SideStore on your iOS device

1

Install AltStore (Temporary)

First, you need to install AltStore temporarily to bootstrap SideStore. Visit altstore.io and download AltServer for your operating system (Windows or macOS).

Note: AltStore is only needed temporarily and can be uninstalled once SideStore is fully set up.

2

Install AltStore on Your iOS Device

Connect your iOS device to your computer via USB, open AltServer, and use it to install AltStore on your device. Make sure iTunes (on Windows) or Finder (on macOS) is installed and can detect your device.

Tip: Enable "Wi-Fi sync" in iTunes/Finder to refresh apps wirelessly later.

3

Download SideStore IPA

Download the latest SideStore IPA file from the official GitHub repository: SideStore Releases

Choose the latest stable release (not pre-release unless you need specific features).

4

Install SideStore Using AltStore

Open AltStore on your iOS device, go to the "My Apps" tab, tap the "+" button in the top left corner, and select the SideStore IPA file you downloaded. AltStore will install SideStore on your device.

This may take a few minutes. Keep your device connected during installation.

5

Install iDevicePair (Required for Wireless Refresh)

For wireless app refreshing, you need to pair your device with your computer using iDevicePair:

  • Windows: Download and install iDeviceInstaller
  • macOS: Install via Homebrew: brew install libimobiledevice
  • Linux: Install via package manager: sudo apt install libimobiledevice6

After installation, connect your device via USB and run: idevicepair pair

6

Configure Anisette Server (Host File Method)

SideStore requires an Anisette server for Apple ID authentication. You have several options:

  • Option A: Use the official SideStore server (may be rate-limited)
  • Option B: Set up your own server using the self-hosting guide
  • Option C: Use a community-hosted server (check SideStore Discord)

To use a custom server, you'll need to modify your hosts file. Add this line to your hosts file:

127.0.0.1 gs.apple.com

Hosts file locations:
• Windows: C:\Windows\System32\drivers\etc\hosts
• macOS/Linux: /etc/hosts

7

Set Up STOS VPN (Alternative Method)

As an alternative to host file modification, you can use STOS VPN to redirect Apple authentication:

  1. Download the STOS VPN configuration from the SideStore GitHub wiki
  2. Import the VPN profile to your device
  3. Enable the VPN when using SideStore
  4. The VPN will handle the Anisette server redirection automatically

This method is easier for most users but requires keeping the VPN active during app refreshes.

STOS VPN Setup Guide
8

Configure SideStore App

Open SideStore on your iOS device and complete the setup:

  • Go to Settings → Anisette Server
  • Select your preferred server method (official, custom, or local)
  • Enter your Apple ID (recommended: use an app-specific password)
  • Enable "Background Refresh" for automatic app updates
  • Configure "Refresh Apps Every" to your preference (24 hours recommended)

Test the configuration by trying to install a simple app first.

9

Add Repositories and Install Apps

Once SideStore is configured and working, you can add repositories and install apps:

  • Use the "Add to SideStore" button below to quickly add this repository
  • Browse available apps in the "Browse" tab
  • Install apps with one tap
  • Manage your installed apps in the "My Apps" tab
10

Final Step: Uninstall AltStore

Once SideStore is fully functional and you've confirmed apps can refresh wirelessly:

  1. Make sure all your important apps are installed and working in SideStore
  2. Test wireless refresh by waiting 24 hours or manually triggering a refresh
  3. Once confirmed working, you can safely uninstall AltStore
  4. Long-press on AltStore and select "Delete App"

Benefits of removing AltStore: Frees up an app slot, reduces complexity, and SideStore operates independently with better wireless refresh capabilities.

Congratulations! You now have a fully independent SideStore setup without AltStore dependencies.
